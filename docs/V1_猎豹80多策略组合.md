## 情绪过滤层（核心保险）

### 情绪指数计算
```
情绪指数 = (上涨家数/总数) × 40 + (涨停家数/100) × 30 + (量比均值/3) × 30
```

### 情绪状态与仓位
| 情绪区间 | 状态 | 操作 | 总仓位上限 |
|----------|------|------|------------|
| 情绪 > 75 | 高位风险 | 禁止开仓，只出不进 | 0% |
| 情绪 55~75 | 正常偏热 | 正常策略执行 | 100% |
| 情绪 40~55 | 正常 | 正常策略执行 | 100% |
| 情绪 30~40 | 偏冷 | 可适当重仓 | 150% |
| 情绪 < 30 | 极度悲观 | 抄底模式，允许超配 | 200% |

## 子策略A：动量突破策略（40%仓位）

### 入场条件（4选3同时满足）
1. **大盘RSI < 45**（超跌反弹区间）
2. **MA5上穿MA20**（短期动量启动）
3. **当日涨幅 3%~9%**（排除涨停一字板）
4. **量比 > 1.5**（资金确认）

### 出场规则
- 止损：-3%
- 第一止盈：+8%（出一半）
- 第二止盈：+15%（出完）
- 移动止损：最大回撤 > -5%

---

## 子策略B：情绪抄底策略（35%仓位）

### 入场条件（4选3同时满足）
1. **情绪指数 < 35**（极度悲观区）
2. **大盘RSI < 30**（超卖）
3. **连续下跌天数 ≥ 3天**
4. **当日成交量 > 昨日1.5倍**（抄底资金入场）

### 出场规则
- 止损：-5%（抄底策略允许更大波动）
- 止盈：+20%（分批：+10%出一半，+20%出完）
- 时间止损：持仓 > 10天无动静，强制出

### 特殊规则
- 情绪 < 30 时，允许单只仓位上限提升至40%
- 连续亏损2次，强制休息3天

---

## 子策略C：政策驱动低估策略（25%仓位）

### 入场条件（3选2同时满足）
1. **政策信号出现**（降准/降息/救市政策发布后3日内）
2. **相关行业RSI < 40**（行业超跌）
3. **北向资金净流入 > 10亿**（外资背书）

### 出场规则
- 止损：-4%
- 止盈：+15%（出一半），+25%（出完）
- 政策兑现：政策发布后第5日无涨幅，强制出

---

## 持仓管理总规则

### 仓位分配
| 子策略 | 最大仓位 | 触发条件 |
|--------|----------|----------|
| 策略A | 40% | 正常情绪 |
| 策略B | 35% | 情绪 < 40时可升至50% |
| 策略C | 25% | 政策信号出现时 |
| **合计上限** | **100%** | 情绪 < 30时可超配至200% |

### 分批建仓
- 底仓30% → 回调确认加30% → 趋势确认补40%
- 最大同期持仓：5只（策略A/B/C各1~2只）

## 年度交易频次保障
- 目标：≥30次/年
- 策略A月均：4次（基础频次）
- 策略B月均：3次（偏左侧，低频）
- 策略C月均：2次（政策触发，不定期）
- 若当月 < 4次：月末最后3天策略A强制选股

## 📊 数据维度清单（OpenClaw开发必需）

### 数据源：akshare（免费）

#### 1. 价格与行情数据
| 字段 | 说明 | 用途 |
|------|------|------|
| date | 日期 | 索引 |
| open/high/low/close | OHLC价格 | 计算MA、RSI |
| volume | 成交量 | 量比计算 |
| amount | 成交额 | 资金流计算 |

#### 2. 市场整体情绪
| 字段 | 说明 | 用途 |
|------|------|------|
| 上涨家数 | 当日上涨股票数 | 情绪指数计算 |
| 下跌家数 | 当日下跌股票数 | 情绪指数计算 |
| 涨停家数 | 当日涨停股票数 | 情绪指数计算 |
| 跌停家数 | 当日跌停股票数 | 风险监控 |
| 炸板家数 | 涨停打开的股数 | 市场情绪确认 |

#### 3. 资金流向数据
| 字段 | 说明 | 用途 |
|------|------|------|
| 北向资金净流入 | 陆股通当日净买入 | 策略C信号 |
| 南向资金净流入 | 港股通当日净买入 | 参考 |
| 主力资金净流入 | 大单净买入额 | 策略A确认 |
| 散户资金净流入 | 小单净买入额 | 反向指标 |

#### 4. 技术指标数据
| 字段 | 说明 | 用途 |
|------|------|------|
| MA5/MA20/MA60 | 均线 | 策略A入场 |
| RSI(14) | 相对强弱 | 入场过滤 |
| MACD | 指数平滑移动平均 | 趋势判断 |
| KDJ | 随机指标 | 策略A入场 |
| 量比 | 当日成交量/5日均量 | 策略A确认 |

#### 5. 政策事件数据
| 字段 | 说明 | 用途 |
|------|------|------|
| 政策日期 | 政策发布日 | 策略C起算 |
| 政策类型 | 降准/降息/救市等 | 分类 |
| 政策力度 | 重大/一般/轻微 | 权重 |
| 受益行业 | 直接受益板块 | 行业筛选 |

#### 6. 股票基础数据
| 字段 | 说明 | 用途 |
|------|------|------|
| 股票代码 | 唯一标识 | 选股池 |
| 股票名称 | 中文名 | 展示 |
| 所属行业 | 申万行业分类 | 策略C行业筛选 |
| ST状态 | 是否ST/退市 | 排除 |
| 涨跌停状态 | 是否涨跌停 | 排除涨停 |

## 回测要求（OpenClaw执行）

### 必须输出指标
1. **年度收益**（每年必须正收益）
2. **年度交易次数**（每年≥30次）
3. **月度交易次数**（每月≥4次）
4. **最大回撤**
5. **夏普比率**
6. **胜率**
7. **各策略贡献收益占比**

### 必须分市场状态统计
- 牛市（情绪 > 55）收益
- 震荡（情绪 40~55）收益
- 熊市（情绪 < 40）收益

### 回测参数
- 滑点：0.3%
- 手续费：万3买卖 + 印花税千1（卖时）
- 排除：ST股、涨跌停无法买入的
