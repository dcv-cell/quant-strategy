## 入场条件（4选3同时满足）
1. **大盘RSI < 45**（超跌反弹区间）
2. **MA5上穿MA20**（短期动量启动）
3. **当日涨幅 3%~9%**（排除涨停一字板，涨停无法买入）
4. **量比 > 1.5**（资金确认）

### 说明
- 4个条件至少满足3个才能开仓
- 条件1和条件2为基础条件，必须满足
- 条件3和条件4至少满足1个

---

## 持仓规则
- 底仓30%，回调加仓30%，趋势确认补仓40%
- 单只仓位上限：总资金30%
- 最大同期持仓：3只

---

## 出场规则
| 类型 | 条件 |
|------|------|
| 止损 | -3%（当日收盘触及即出） |
| 第一止盈 | +8%（出一半） |
| 第二止盈 | +15%（出完） |
| 移动止损 | 持仓期间最大回撤 > -5% 强制出 |

---

## 市场状态过滤（核心保命机制）

```python
# 市场状态判断伪代码
IF 大盘_MA20向下趋势 AND 情绪指数 > 70:
    禁止开仓，只出不进  # 熊市高位，不交易
    
IF 大盘连续下跌 > 5天:
    仓位降至1只，轻仓操作
    
IF 当月交易次数 < 4:
    月末最后3天强制选股建仓  # 保频次
    
IF 年度收益已达 +80%:
    停止新开仓，本年度不再交易  # 达标不贪
    
IF 年度回撤 > 10%:
    启动保本模式，只做ETF不动股
```

## 年度交易频次保障
- 每年目标：≥30次买卖操作
- 每月目标：≥4次
- 若当月 < 4次操作：月末最后3天强制选股建仓
- 每年12月最后5个交易日：年度清仓，不持仓过夜

## 5年连续正收益保障机制

### 保命规则（必须严格执行）
1. **熊市仓位上限**：单只≤10%，总仓位≤30%
2. **连续亏损2次**：强制休息3天，不追票
3. **年度回撤 > 10%**：切换保本模式（只做指数ETF）
4. **年度收益 > 80%**：立即停止新开仓，不博弈最后收益

### 风险控制优先级
```
止损 > 保本 > 频次 > 收益
```
